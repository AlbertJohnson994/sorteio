# Sorteio

Este projeto é uma página simples de sorteio desenvolvida em HTML.

## Como usar

1. Abra o arquivo `index.html` em seu navegador.
2. Siga as instruções na página para participar do sorteio.

## Estrutura do projeto

- `index.html`: Página principal do sorteio.

## Sobre

Projeto criado para fins de estudo na EBAC.
